This data is scraped from the Bexar County, Texas ArcGIS map interface for COVID-19 case counts (https://cosagis.maps.arcgis.com/apps/opsdashboard/index.html#/d2c7584fe9fd4da1b30cb9d6cc311163); data is reported by the San Antonio Metropolitan Health District.
It includes case counts by zip code in Bexar County, Texas.
Note: currently data for zip code 78108 is problematic.
This file is generated by bexar-county_scrape.py in the scripts folder. Note that it requires population by zip code data, since only case proportions per 100k are reported.

This file contains scraped data from the COVID-19 map for Washtenaw County, Michigan. 
It contains case counts over time for Washtenaw by zip code (dashboard: 
https://washtenaw.maps.arcgis.com/apps/opsdashboard/index.html#/f3d49937af114e58b11749722076921d).
The CSV file is generated by pull_arcgis_data.py, which should be run once a day.

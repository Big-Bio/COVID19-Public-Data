This file contains scraped data from the COVID-19 map for St. Louis County, Missouri. 
It contains case counts over time for St. Louis by zip code (dashboard: 
https://www.stlouis-mo.gov/covid-19/data/zip.cfm).
Note that this data is not entirely comprehensive since the county website states, 
"Data for some zip codes has been suppressed in accordance with patient privacy 
protections. Zip Codes with fewer than 5 cases are not displayed."
The CSV file is generated by pull_arcgis_data.py, which should be run once a day.

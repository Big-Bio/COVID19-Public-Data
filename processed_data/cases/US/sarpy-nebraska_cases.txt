This file contains scraped data from the COVID-19 map for Sarpy County, Nebrasksa. 
It contains case counts over time for Sarpy by zip code (dashboard: 
https://covid19.dogis.org/app/90ea18d2368e43d495255a13b6989ae8).
The CSV file is generated by pull_arcgis_data.py, which should be run once a day.
